# Future_ML_03-Resume-Candidate-Screening-System
📄 Resume Screening System using NLP & Machine Learning

🔹 Project Overview

This project is a Resume Screening System that automatically analyzes and ranks resumes based on their relevance to a given job description. It uses Natural Language Processing (NLP) and Machine Learning techniques to evaluate candidates efficiently.

---

🔹 Features

- 📌 Resume text cleaning and preprocessing
- 📌 Skill extraction from resumes
- 📌 Matching resumes with job description
- 📌 Candidate ranking based on similarity score
- 📌 Skill gap identification

---

🔹 Technologies Used

- Python
- Scikit-learn
- NLP (Text Processing)
- TF-IDF Vectorization
- Cosine Similarity

---

🔹 How It Works

1. Reads multiple resumes from a text file
2. Cleans and preprocesses text data
3. Converts text into numerical vectors using TF-IDF
4. Compares resumes with job description
5. Calculates similarity scores
6. Ranks candidates based on relevance
7. Identifies missing skills
